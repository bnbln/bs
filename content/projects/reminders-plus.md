Some productivity apps are *too good* at letting you plan. You end up with perfect lists… and zero momentum.

**Reminders+** is my attempt to fix that with one simple rule: **do at least one thing every day** — and let the streak do the heavy lifting. It’s a clean, fast, local-first iOS app we designed together with **Max Möckel** and built in **React Native**.  [App Store](https://apps.apple.com/ng/app/reminders/id6758573672)


# The idea: “one task a day” beats “a perfect plan”

Instead of pushing you into huge lists, Reminders+ is intentionally built around consistency:

- **Tasks** for the stuff you need to get done
- **Routines** for habits (daily / weekly / monthly)
- **Projects** for bigger goals
- **Streaks** as the motivator — your progress stays visible and tangible

That mechanic sounds small, but it changes how you use the app: you stop aiming for “finish everything” and start aiming for “show up today”.

```insight title="Motivation without gamification noise"
The streak isn’t a badge explosion — it’s just a calm, persistent signal: “you’re building something.” The UI stays minimal so the feedback loop stays strong.
```

# Calm, modern, and focused UX/UI Design

We designed Reminders+ around a few principles:
- Fast capture (getting a task in should never feel like work)
- Clear hierarchy between Tasks / Routines / Projects
- A UI that stays out of the way, so your attention stays on doing, not organizing

# Development

So Reminders+ is built on React Native and local-first by design:
- 100% local storage, no cloud dependency
- “Data not collected” (App Store privacy)
- Fast & lightweight, with smooth navigation and animations
